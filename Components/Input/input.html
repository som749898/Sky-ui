Inputs

Inputs are necessary for user interaction with the website. Inputs are used to take information from the user, and then save the data to the server, accordingly.

Label Inputs

Label Input has a label present at the top of the input field.


          <div class="input-group">
            <label class="mg-bottom-1x" for="username">Username *</label>
            <input placeholder="Please enter your username" class="py-1x" type="text">
          </div>

Form validation styles

The Input component are used to get user input in a text field.They can be customized to show error error-message, and change color accordingly.

Wrong Password. Try again. 

          <div class="input-group input-error">
            <label class="mg-bottom-1x" for="password">Password *</label>
            <input class="py-1x mg-bottom-1x" type="text">
            <div class="err-message">
              Wrong Password. Try again.
            </div>

Radio

Radio buttons allow the user to select one option from a set.

Select one of the following


          <p class="mg-bottom-1x">Select one of the following

          <div class="flex vertical-center">
            <input type="radio" name="checked" id="checked" checked>
          <label class="mg-right-1x" for="checked">Checked</label>
          <input type="radio" name="unchecked" id="unchecked">
          <label class="mg-right-1x" for="unchecked">Unchecked</label>
          <input type="radio" name="disabled" id="disabled" disabled>
          <label class="mg-right-1x" for="disabled">Disabled</label>
          </div>

Checkbox

Checkboxes allow the user to select one or more items from a set. Checkboxes can be used to turn an option on or off.

Please select all the correct option


          <p class="mg-bottom-1x">Please select all the correct option</p>
          <div class="flex vertical-center">
            <input type="checkbox" name="checked" id="checked" checked>
            <label class="mg-right-1x" for="checked">Checked</label>
            <input type="checkbox" name="unchecked" id="unchecked">
            <label class="mg-right-1x" for="unchecked">Unchecked</label>
            <input type="checkbox" name="disabled" id="disabled" disabled>
            <label class="mg-right-1x" for="disabled">Disabled</label>
          </div>
