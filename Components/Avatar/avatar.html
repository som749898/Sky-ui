Avatars

Avatar can be used to show user's profile picture on profile information page, on navigation bar, in blogs and grid items.

Profile Avatar

Profile Avatars can be created using the class 'avatar', inside the img tag.


          <img class="avatar" src="..." alt="Demon Slayer">
          <img class="avatar" src="..." alt="Demon Slayer">
          <img class="avatar" src="..." alt="Demon Slayer">

Circular Avatars

Size of the avatar can be changed by adding lg (11rem) | md (9rem) | sm (5rem) class to existing avatar class


          <img class="avatar avatar-lg" src="..." alt="Demon Slayer">
          <img class="avatar avatar-md" src="..." alt="Demon Slayer">
          <img class="avatar avatar-sm" src="..." alt="Demon Slayer">

Sqaure Avatars

Addind a 'square' class to existing class, will generate a square shaped avatar.


          <img class="avatar-square avatar-lg" src="..." alt="Demon Slayer">
          <img class="avatar-square avatar-md" src="..." alt="Demon Slayer">
          <img class="avatar-square avatar-sm" src="..." alt="Demon Slayer">

Avatars with Status

The avatar component is typically used to display square/circle user profile pictures along with status indicators. There are typically 3 status indicators: online, offline and busy.


          <div class="avatar-online">
            <img class="avatar" src="/assets/img/avatar/Tanjiro__Demon_Slayer_.png" alt="Demon Slayer">
            <span class="status online"></span>
          </div>
          <div class="avatar-offline">
            <img class="avatar" src="/assets/img/avatar/Tanjiro__Demon_Slayer_.png" alt="Demon Slayer">
            <span class="status offline"></span>
          </div>
          <div class="avatar-busy">
            <img class="avatar" src="/assets/img/avatar/Tanjiro__Demon_Slayer_.png" alt="Demon Slayer">
