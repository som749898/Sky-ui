Text Utilities

Documentation and examples for common text utilities to control alignment, sizing, colored text.

Text Sizes

Text Sizes give your the freedom of chosing what kind of size you want in your project for diffrent types of text's.

This is an h1 heading
This is an h2 heading
This is an h3 heading
This is an h4 heading

          <div class="heading1">This is an h1 heading</div>
          <div class="heading2">This is an h2 heading</div>
          <div class="heading3">This is an h3 heading</div>
          <div class="heading4">This is an h4 heading</div>

Small Text

For small text you can use the class names txt-small and txt-xs.

This is a small text.

This is an extra small text.


          <p class="text-small">This is a small text.</p>
          <p class="text-xs">This is an extra small text.</p>
